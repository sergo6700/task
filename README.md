## About Project

- Laravel 7 / Vue Js.
- Timon JWT.
- VueX.
- i18n.
- Sweet Alert.
- Vee Validate

## About Functionality

- Login, Register, Remember Me, Forgot Password, Email Verification.
- User Settings: Change Password, Change Profile.
- Localization: EN/RUS.
- CRUD For Resume.




For run project open terminal and type

composer install

composer dump-autoload

npm install

npm run dev(or watch)

php artisan key:generate

php artisan  jwt:secret 

php artisan queue:listen 

php artisan queue:watch --queue=email
